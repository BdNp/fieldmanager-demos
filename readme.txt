This is a demo of Fieldmanager

== Description ==

A plugin to demonstrate [Fieldmanager](http://fieldmanager.org). Requires Fieldmanager, of course.

== Installation ==

1. Upload plugin to the \`/wp-content/plugins/\` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
